# CiclePark

A **React Native (Expo)** app to help urban cyclists decide **where to park their bike safely**: map, spot discovery, and a foundation for contributions (MVP in progress).

## Requirements

- Node.js (LTS recommended)
- [Yarn](https://yarnpkg.com/) or npm
- For native builds: Android Studio / Xcode as needed per platform

## Quick start

```bash
yarn install
yarn start
```

- **Dev client** development: `yarn start` (see `package.json`).
- Expo Go: `yarn start:go`.
